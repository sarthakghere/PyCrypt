PyCrypt

Enhance the security of your files through encryption. Begin by generating your encryption keys.

Encrypt a File

To encrypt a file, upload the receiver's public key along with the file. The file can only be decrypted by the intended recipient using their private key.

Decrypt a File

To decrypt a received file, upload your private key that corresponds to the public key shared with the sender.

Important: It is the sole responsibility of the user to secure their encryption keys. Users' keys are not stored anywhere on the server.
